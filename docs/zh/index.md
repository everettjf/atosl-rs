# 中文文档

`atosl` 是一个自包含的单文件命令行工具，用于将地址符号化（symbolication）——把崩溃报告中的地址解析为函数名与源码位置。它直接读取 Mach-O、DWARF 与 ELF，无需依赖 Xcode 或 Apple 工具链。
